# 免费图书分享 — 开发文档

## 概述

452 本免费电子书，16 个分类，数据在七牛云 CDN。做一个小程序供用户浏览、搜索、复制百度网盘下载链接。后续适配 H5。

## 技术栈

Taro 4 + React 18 + TypeScript + NutUI-React + SCSS

编译目标：weapp（先）→ h5（后）

## 数据

全部在七牛云 CDN，前端直接请求，无后端。

```
CDN: http://qn-cdn.windlliu.com/books-tidy/
├── configs.json    ← 各分类 JSON 的 URL 入口
├── configs/index.json    ← [{type, count}, ...] 分类汇总
├── configs/历史.json      ← 该分类书籍数组
├── configs/文学.json
└── ...
```

**单本书结构：**
```json
{
  "id": "uuid",
  "name": "书名",
  "author": "作者",
  "type": "历史",
  "bd_link": "https://pan.baidu.com/s/xxx?pwd=xxx",
  "picUrl": "https://xxx/{id}.jpg",
  "brief": "简介"
}
```

**加载策略：** 启动时加载 configs + index → 点某分类才加载该分类 JSON → 缓存到内存

## 页面（3 个）

**首页** — 搜索栏 + 分类卡片 Grid（分类名 + 数量），点分类进列表页

**分类列表页** — 书籍卡片列表（封面缩略图 + 书名 + 作者 + 简介），上拉分页（每页 20，前端切片），点卡片进详情页

**书籍详情页** — 大图 + 书名/作者/分类 + 完整简介 + 复制链接按钮

## 核心功能

- **搜索：** 书名/作者模糊匹配，首次搜索时批量加载所有分类数据，后续走缓存，300ms 防抖
- **复制链接：** `Taro.setClipboardData`，解析 `pwd=` 提取码，复制格式 `链接：xxx\n提取码：xxx`
- **图片优化：** 列表用 `{picUrl}?imageView2/2/w/200/h/280/q/85`，详情用 w/400
- **数据缓存：** 内存 Map 缓存已加载分类

## 开发步骤

1. `taro init` 初始化，配 AppID，装 NutUI-React
2. 数据请求层 + 常量配置
3. 首页（分类 Grid + 搜索栏）
4. 分类列表页（卡片 + 分页）
5. 详情页（大图 + 复制链接）
6. 搜索功能（跨分类加载 + 过滤）
7. 优化：骨架屏、错误处理、缓存

## 注意

- 七牛云测试域名 `qn-cdn.windlliu.com` 不能用于小程序正式上线，需绑定已备案域名
- 开发阶段勾选"不校验合法域名"即可
- 小程序审核用"复制链接"方式比直接跳转更安全
- 图片务必用缩略图参数，这是主要性能瓶颈
