## 1. 开发上下文

**技术栈**：Taro 4 + React 18 + TypeScript + SCSS + NutUI React Taro
**关联模块**：详情页（`src/pages/detail/index.tsx`）、剪贴板工具（`src/utils/clipboard.ts`）
**现有 UI 规范**：参考 `docs/ui-ux-spec.md`，操作按钮高度 88rpx 胶囊形，Toast 2s 自动消失，弹窗阴影 `0 4px 20px rgba(0,0,0,0.12)`，圆角 12rpx

## 2. 验收标准

- [ ] 点击「复制下载链接」后，链接复制到剪贴板，并弹出底部引导面板
- [ ] 引导面板展示已复制的链接文本（链接 + 提取码，格式与当前一致）
- [ ] 引导面板包含 3 步操作指引：切换到百度网盘 → 粘贴链接 → 输入提取码
- [ ] 点击面板外部或「知道了」按钮可关闭面板
- [ ] 复制失败的错误场景仍正常处理（Toast 提示）
- [ ] 收藏、分享等其他现有功能未受影响
- [ ] 面板样式与现有 UI 规范一致（配色、圆角、字体）

### 3.3 UI 改动

- 修改页面：详情页（`src/pages/detail/index`）— 新增复制成功引导面板（底部弹出式遮罩面板）
- 复用样式：使用 `variables.scss` 中的 `$color-primary`、`$radius-lg`、`$spacing-*`、`$shadow-modal`
- 参考风格：现有 `ErrorView` 的居中卡片式布局，但改为底部弹出

**引导面板结构**：

```
┌──────────────────────────────┐
│       半透明遮罩 (overlay)      │
│  ┌──────────────────────────┐│
│  │  ✓ 链接已复制到剪贴板      ││
│  │                          ││
│  │  ┌────────────────────┐  ││
│  │  │ 链接：https://...   │  ││
│  │  │ 提取码：xxxx        │  ││
│  │  └────────────────────┘  ││
│  │                          ││
│  │  接下来：                 ││
│  │  1. 打开「百度网盘」App   ││
│  │  2. 粘贴分享链接          ││
│  │  3. 输入提取码（如有）     ││
│  │                          ││
│  │  ┌───────────────────┐   ││
│  │  │     知道了         │   ││
│  │  └───────────────────┘   ││
│  └──────────────────────────┘│
└──────────────────────────────┘
```
